# Proyecto Ecommerce pwa UTN
Este proyecto pertenece al curso Programador web avanzado que dicta la UTN BA. En este mismo, decidí construir un Ecommerce.
## Tecnologías utilizadas 
* NodeJs
* Framework Express
* MySql
* Angular
* Angular Material
* TypeScript
* JavaScrpit
